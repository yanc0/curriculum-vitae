Latest Customer migrations

Jun. 2018 - Dec 2018

They were previously hosted on a classic bare metal datacenter and we moved all their workloads to GCP. Now the platform is fully managed by Google Cloud and SKALE-5, enhancing their products is way easier now.

It is a pretty young product but I have lots of ideas for making the new Hakernews for DevOps. I think this website reflects my personnality:

Beeping

2016
Before the advent of Prometheus, our monitoring system had the need of a simple, distributed HTTP(s) checks as a service, so I developed Beeping It received a warm welcome by the community. It was my first real opensource project and I was really glad to merge pull requests coming from all over the world. I don't maintain this project anymore, the new prometheus blackbox-exporter is now way more standard.

Greedee

2017
Greedee was made for performance and modularity. It is a small HTTP server that receives, transforms and stores metrics in timeseries backend (graphite). It still runs in production for legacy systems that are monitored by collectd agents. I learned many things in Go with this project but no one will use this software, it is too much specialized and for old ugly things :)
